# **App Name**: HomeView

## Core Features:

- Interactive Map: Display an interactive map using Leaflet or similar library.
- House Information: Allow users to select a house on the map and view details like address, size, number of rooms, year of construction, wall material, building series, number of floors, elevator availability, floor type, foundation type, garbage chute availability, and hot water supply system.
- Floor Plan Display: Display the floor plan of the selected house with interactive elements. Allows zooming and panning.
- Property Search: Enable users to search for properties by address, price range, or other criteria.
- Data Storage: Store house information, floor plans, and location data. Store images on a cloud storage service.
- Cadastral Data Loading: Load house data from the cadastral map.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional and trustworthy feel.
- Background color: Light gray (#F5F5F5) to provide a clean and neutral backdrop.
- Accent color: Teal (#009688) for interactive elements and call-to-action buttons.
- Font pairing: 'Inter' for body text, and 'Space Grotesk' for headlines; both are sans-serif fonts with a modern and clean appearance.
- Use simple, modern icons to represent different features and actions.
- Use a clean and intuitive layout with a clear hierarchy of information.
- Subtle animations when loading house details and transitioning between floor plan views.
